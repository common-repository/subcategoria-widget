Permite voc&ecirc; exibir subcategorias de uma determinada categoria no Widget

== Description ==

Subcategoria Widget possibilita voc&ecirc; exibir widgets onde cada widget vai exibir somente subcategorias de uma determinada categoria. Note que os recursos do widget categorias padr&atilde;o tamb&eacute;m est&atilde;o dispon&iacute;veis. Al&eacute;m disso, voc&ecirc; tamb&eacute;m pode definir se subcategorias que n&atilde;o tenham posts devem ou n&atilde;o ser exibidas no widget.

Recursos:

* Op&ccedil;&atilde;o para configurar o t&iacute;tulo do Widget
* Exibir/Ocultar subcategorias que n&atilde;o possuem posts
* Defini&ccedil;&atilde;o da categoria pai &agrave; qual o widget ir&aacute; exibir os filhos
* Ativar/Desativar exibi&ccedil;&atilde;o da quantia de posts uma subcategoria possui
* Ativar/Desativar exibi&ccedil;&atilde;o das subcategorias de forma hier&aacute;rquica

== Installation ==

1. Baixe o plugin.
2. Envie o plugin para a pasta de plugins de seu blog, geralmente &eacute; a pasta wp-content/plugins
3. V&aacute; na sess&atilde;o de plugins no painel de administra&ccedil;&atilde;o do Wordpress e ative o plugin
4. V&aacute; na sess&atilde;o de Widgets e configure o widget &quot;Subcategoria Widget&quot; como desejar
